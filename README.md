Automation
- The file table above is generated by `scripts/updateReadme.js`, which scans the `CodeFiles/` folder and rewrites the table with GitHub links (defaults to the `main` branch and the repo `https://github.com/NothingADSR123/JS_Mastery`).
- A custom `pre-push` hook (stored in `.githooks/pre-push`) keeps the README synchronized automatically before every `git push`. Set it up once with:
	```bash
	git config core.hooksPath .githooks
	```
